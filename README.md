# Omniversify Tifinagh Dictionary API

A REST API for Tifinagh script transliteration and Amazigh dictionary lookup. Built with [Bun](https://bun.sh) + [Hono](https://hono.dev).

## Features

- **Transliteration** — convert between Arabic, Latin, and Tifinagh scripts
- **Dictionary lookup** — search for word meanings across Arabic, English, and Tamazight
- **Auto-detection** — automatically detects input script when not specified
- **Fuzzy search** — matches exact, prefix, and contains queries

## Prerequisites

- [Bun](https://bun.sh) >= 1.0

## Quick Start

```bash
# Install dependencies
bun install

# Download the dictionary dataset (required for /translate)
bun run download-dataset

# Start the server
bun run start
```

The server will start on `http://localhost:3001` (or `$PORT` if set).

## Endpoints

### `GET /help`

Returns a JSON description of all available endpoints with examples.

### `POST /retype`

Convert text between Arabic, Latin, and Tifinagh scripts.

**Request body:**

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `text` | string | yes | The text to convert |
| `from` | string | no | Source script: `"arabic"`, `"latin"`, or `"tifinagh"` (auto-detected if omitted) |
| `to` | string | no | Target script: `"arabic"`, `"latin"`, or `"tifinagh"` (auto-detected if omitted) |

**Examples:**

```bash
# Arabic → Tifinagh
curl -X POST http://localhost:3001/retype \
  -H "Content-Type: application/json" \
  -d '{"text": "مرحبا", "from": "arabic", "to": "tifinagh"}'

# Response:
# {
#   "input": { "text": "مرحبا", "script": "arabic" },
#   "output": { "text": "ⵎⵔⵃⴱⴰ", "script": "tifinagh" }
# }

# Tifinagh → Latin (auto-detect)
curl -X POST http://localhost:3001/retype \
  -H "Content-Type: application/json" \
  -d '{"text": "ⵜⴰⵎⴰⵣⵉⵖⵜ"}'

# Response:
# {
#   "input": { "text": "ⵜⴰⵎⴰⵣⵉⵖⵜ", "script": "auto-detected" },
#   "output": { "text": "tamaziɣt", "script": "auto-detected" }
# }

# Latin → Tifinagh
curl -X POST http://localhost:3001/retype \
  -H "Content-Type: application/json" \
  -d '{"text": "anẓar", "from": "latin", "to": "tifinagh"}'

# Arabic → Latin (via Tifinagh intermediate)
curl -X POST http://localhost:3001/retype \
  -H "Content-Type: application/json" \
  -d '{"text": "مطر", "from": "arabic", "to": "latin"}'
```

### `POST /translate`

Look up a word in the Amazigh dictionary.

**Request body:**

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `text` | string | yes | The word to look up |
| `from` | string | no | Source language: `"arabic"`, `"english"`, `"tifinagh"`, or `"auto"` (default) |

**Examples:**

```bash
# Look up an Arabic word
curl -X POST http://localhost:3001/translate \
  -H "Content-Type: application/json" \
  -d '{"text": "مطر", "from": "arabic"}'

# Response:
# {
#   "query": "مطر",
#   "results": [
#     {
#       "word": "ⴰⵏⵥⴰⵔ",
#       "pronunciation": "anẓar",
#       "type": "",
#       "arabic": "مطر",
#       "english": "rain"
#     }
#   ]
# }

# Look up an English word
curl -X POST http://localhost:3001/translate \
  -H "Content-Type: application/json" \
  -d '{"text": "rain", "from": "english"}'

# Auto-detect (works for Arabic, English, and Tifinagh)
curl -X POST http://localhost:3001/translate \
  -H "Content-Type: application/json" \
  -d '{"text": "ⴰⵏⵥⴰⵔ"}'
```

**Note:** The `/translate` endpoint requires the dictionary dataset to be downloaded first. Run `bun run download-dataset` before starting the server.

## Scripts

| Script | Description |
|--------|-------------|
| `bun run start` | Start the server |
| `bun run dev` | Start with hot reload (`--watch`) |
| `bun run download-dataset` | Download dictionary datasets from Hugging Face |

## Project Structure

```
src/
├── index.ts         # Hono app entry — routes, server
├── retype.ts        # Script transliteration engine
├── translate.ts     # (unused; logic in index.ts)
└── dictionary.ts    # Dataset loader + search index

scripts/
└── download-dataset.ts   # Fetches and merges datasets

data/
└── dictionary.json  # Generated by download script (gitignored)
```

## Dataset Sources

This API uses two datasets from Hugging Face:

1. **[amawal-dataset](https://huggingface.co/datasets/prothmane/amawal-dataset)** (CC-BY-4.0) — 15K Amazigh words with Arabic translations and Latin pronunciation
2. **[ircam-dglai-dataset](https://huggingface.co/datasets/abdelhaqueidali/ircam-dglai-dataset)** — 17K entries from the IRCAM DGLAi dictionary with Arabic and English translations (AI-augmented)

Only Arabic and English are loaded; entries are de-duplicated by Tifinagh word.
